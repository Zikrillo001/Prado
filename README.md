# Prado — Loyiha ish papkasi

Dilshod Prado uchun **premium yo'ltanmas jamoa + marketplace** loyihasi.

## Bu papka nima
Loyihaning barcha ma'lumotlari, tahlili, rejasi va ish materiallari bir joyda.
Kod loyihasi hali yaratilmagan — u keyinchalik `project/` ichida bo'ladi.

## Papka tuzilishi
- `docs/` — tahlil, reja, prezentatsiya tahlili, funksiya inventari
- `planning/` — ochiq savollar, qarorlar jurnali, MVP qamrovi
- `research/` — qo'shimcha research uchun (rezerv)
- `assets/` — original prezentatsiya + reference mockup rasmlar
- `brand/` — brend yo'nalishi (rang, logo, nom)
- `project/` — kelajakdagi kod loyihasi shu yerda bo'ladi (hozircha bo'sh)

## Joriy holat
- Tahlil va strategiya: **TUGADI**
- Model: klassifayd vositachilik, freemium, nisha + ishonch + jamoa
- Reja: **Blueprint v1 tayyor** (`docs/02-reja-blueprint.md`)
- Keyingi qadam: Dilshoddan raqamlar/qarorlar -> MVP detallash -> kod loyihasi
