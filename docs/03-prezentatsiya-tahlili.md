# Prezentatsiya tahlili (Ramazon deki)

Boshlang'ich dizayn deki (12 slayd, asosan mockup). Bu yerda uning tahlili va qaror.

## Dekda nima bor
Aslida uchta mahsulot bitta dekka qo'shilgan:
1. **DILSHOD_PRADO marketplace** — OLX uslubidagi e'lon platformasi, tarif bo'yicha pullik e'lon (1/3/5/7 kun, 10-50 ming so'm), admin panel, Telegram login, to'lov.
2. **PRADO CLUB** — jamoa/klub, tadbirlar, chipta, a'zolik (Gold), a'zo profillari.
3. **Ehtiyot qismlar marketpleysi** — alohida bo'lim.

## Kuchli tomonlari
- Vizual identifikatsiya (qora + oltin, premium yo'ltanmas kayfiyati) — professional.
- Club g'oyasi — ironik tarzda ko'milib ketgan, lekin Dilshodning haqiqiy aktiviga mos.
- Ishonch belgilari (rasmiy do'kon, tekshirilgan sotuvchi) — to'g'ri instinkt.
- Umuman ko'p narsa o'ylangan (oqim, tarif, to'lov, qism, admin).

## Nima yetishmaydi (haqiqiy muammolar)
- **Muammo -> yechim emas, ekran -> funksiya deki.** "Sayt qanday ko'rinadi" ga javob beradi, "qaysi muammoni yechadi va ishlaydimi" ga emas.
- Sovuq start (tovuq-tuxum) muammosi ko'rilmagan.
- Soxta raqamlar (12 458 foydalanuvchi) real qilib ko'rsatilgan.
- "Har e'lon pullik" model — research bo'yicha bepul raqib turganida ishlamaydi.
- Bir odam bo'g'iniga tegmaydi — aksincha ish yukini oshiradi.
- Brend nomuvofiqligi (Prado Club vs Dilshod_Prado).

## Qaror (keep / replace / drop)
- **Saqlaymiz:** vizual identifikatsiya, Club g'oyasi, ambitsiya, ishonch belgilari.
- **Almashtiramiz:** anonim OLX-nusxa -> ishonchga langarli model; soxta raqamlar -> real go-to-market; "har e'lon pullik" -> freemium.
- **Kechiktiramiz:** ehtiyot qism va servis hamkorligini ishga tushirish qamrovidan (P2).

Batafsil funksiya qarori: `04-funksiya-inventari.md`.
