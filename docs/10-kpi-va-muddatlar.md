# KPI va muddatlar

## Muvaffaqiyat ko'rsatkichlari (KPI)
- Faol e'lonlar soni
- Oylik faol foydalanuvchi (MAU)
- E'lon -> sotuvchiga bog'lanish konversiyasi (Telegram/qo'ng'iroq bosish)
- Ko'tarish (VIP/TOP) to'lovlari soni va daromadi
- Sotuvchi qaytishi (takroriy e'lon)
- Sotilgan e'lonlar (aylanma tezligi — hozir 1 haftaga bormaydi)
- Organik trafik (Google'dan kirish — SEO natijasi)

## Taxminiy muddatlar (byudjet ijobiy deb)
- **Oy 1-2:** dizayn (UI) + MVP dev boshlanishi + yuridik rasmiylashtirish
- **Oy 3-4:** MVP tayyor + yopiq beta + inventar to'ldirish
- **Oy 5:** ochiq launch + auditoriya ko'chirish kampaniyasi
- **Oy 6-8:** Club (Faza 1) + promo chuqurligi + retention
- **Oy 9-12:** ehtiyot qism + diler paketlari + hamkorlik (Faza 2)

*Bu skeleton muddat — aniq reja jamoa hajmi va byudjetga qarab.*
