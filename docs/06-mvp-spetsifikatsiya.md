# MVP spetsifikatsiyasi (ekran + oqim)

MVP modullari, har biri ekran va oqim darajasida. Freemium: e'lon bepul, ko'tarish pullik.

## 1. Bosh sahifa (landing)
- Hero: qidiruv paneli + "Siz orzu qilgan Pradoni bizda toping"
- Kategoriyalar qatori (marka bo'yicha)
- TOP / VIP e'lonlar bloki (tepada)
- Yangi e'lonlar
- Ishonch bloki: "Dilshod Prado — tekshirilgan sotuvchilar, ishonchli savdo"
- "Qanday ishlaydi" (3 qadam)
- SEO: har blok server-rendered (Google'da ko'rinish uchun)

## 2. Katalog / ro'yxat
- Chap: filtr paneli (marka, model, yil, probeg, narx, yoqilg'i, uzatma, holat, hudud)
- E'lon kartochkalari (rasm, narx, asosiy xususiyat, joy, VIP/TOP belgisi)
- Saralash (yangi, arzon, qimmat), VIP/TOP tepada
- Sahifalash / cheksiz skroll

## 3. E'lon detali
- Rasm galereyasi
- Narx + VIP/TOP belgisi + holat
- Xususiyatlar jadvali (yil, probeg, dvigatel, uzatma, yoqilg'i, rang, komplektatsiya)
- Xususiyat ikonlari (4WD, 360 kamera, lyuk, kruiz, start/stop, elektr o'rindiq)
- Tavsif
- Sotuvchi kartasi: ism, tekshirilgan belgi, "Telegram yuborish" + "Qo'ng'iroq qilish"
- Sevimliga qo'shish, ulashish
- O'xshash e'lonlar

## 4. E'lon joylash oqimi (4 bosqich)
1. Avtomobil ma'lumoti: Marka, Model, Yil, Motor, Yoqilg'i, Uzatma
2. Qo'shimcha: Probeg, Rang, Holat, Joylashuv, Narx + Tavsif
3. Rasmlar yuklash (raqamni yashirish imkoni — keyin)
4. E'lon bepul joylanadi -> ixtiyoriy ko'tarish (VIP/TOP) tanlash -> to'lov
- Yakunda: moderatsiyaga tushadi -> tasdiqlangach faol bo'ladi

## 5. Profil + Mening e'lonlarim
- Tablar: Faol / Yakunlanadi / Arxiv
- Har e'lon: tahrirlash, ko'tarish (VIP/TOP), o'chirish, "Davom ettirish"
- Sevimlilar, Saqlangan qidiruvlar
- Sotuvchi holati: oddiy / tekshirilgan / rasmiy do'kon

## 6. Autentifikatsiya
- Telegram orqali kirish (login widget / bot OTP)
- Muqobil: telefon raqami + SMS OTP

## 7. Ishonch / verifikatsiya
- Sotuvchi "tekshirilgan" bo'lish arizasi
- "Rasmiy do'kon" arizasi (Dilshod Prado va ishonchli dilerlar uchun)
- Admin tasdig'i orqali belgi beriladi

## 8. Admin panel (MVP asosiy)
- Dashboard: e'lon/foydalanuvchi/to'lov statistikasi, grafiklar, geografiya
- E'lonlar: moderatsiya (tasdiqlash / rad / arxiv), tahrirlash
- Foydalanuvchilar: ko'rish, tekshirilgan belgi berish, bloklash
- To'lovlar: ro'yxat, holat
- Kategoriyalar, Tariflar boshqaruvi
- Shikoyatlar (report) ko'rish

## 9. To'lov (faqat ko'tarish haqi)
- Usul tanlash: Payme / Click
- Tasdiqlash -> ko'tarish avto-faollashadi -> bildirishnoma -> statistikaga qo'shiladi
