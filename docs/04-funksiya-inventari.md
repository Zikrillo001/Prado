# Funksiya inventari (dekdan)

Egangiz tasdiqlagan funksiyalar, strategiyaga qarab belgilangan.
Belgi: [OLAMIZ] yadro · [ISHONCH] farqlash · [KEYIN] keyingi bosqich · [QAYTA] qayta o'ylash.

## [ISHONCH] Ishonch + jamoa (eng aqlli instinktlar)
- Tekshirilgan sotuvchi belgilari (Rasmiy do'kon, Ishonchli sotuvchi)
- Telegram orqali kirish + telefon tasdiqlash
- Admin moderatsiya (e'lonni tasdiqlash)
- Club bloki: tadbirlar, chipta, a'zolik (Gold + ballar), galereya, a'zo profillari, sharhlar, kalendar

## [OLAMIZ] Yadro marketplace + avtomatlashtirish
- E'lon hayot sikli: Faol -> Yakunlanish arafasida -> Bugun yakunlanadi -> Arxiv (avtomatik) + "Davom ettirish"
- 4-bosqichli e'lon joylash oqimi
- E'lon detali: xususiyat ikonlari (4WD, 360 kamera, lyuk, kruiz, start/stop, elektr o'rindiq) + Telegram/qo'ng'iroq bog'lanish
- Profil + "Mening e'lonlarim" (Faol/Yakunlanadi/Arxiv), Sevimlilar, saqlangan qidiruv
- Admin panel: dashboard (statistika, grafik, geografiya, faoliyat), e'lon/foydalanuvchi/to'lov boshqaruvi, moderatsiya, eksport
- To'lovda e'lon avtomatik faollashishi

## [OLAMIZ] To'lov (faqat ko'tarish haqi uchun)
- Payme / Click / Uzum / Visa / HUMO, avto-faollashish, bildirishnoma

## [KEYIN] Keyinroq (qamrov/bosqich)
- Ehtiyot qismlar marketpleysi (Avtoelon'da bor — farqlash emas)
- Servis hamkorligi + a'zolarga chegirma
- Bannerlar (reklama), Retush (raqam yashirish), refund

## [QAYTA] Qayta o'ylash (tashlamaymiz)
- Tarif rejalari: har e'lon pullik -> **freemium** (e'lon bepul, VIP/TOP/ko'tarish pullik). VIP/TOP tushunchasi qoladi.
- Daromad xilma-xilligi (e'lon, VIP, TOP, chipta, a'zolik, reklama) — yaxshi, saqlaymiz; faqat asosiy e'lon bepul.

## Xulosa
Funksiyalar asosan olinadi. O'zgaradigani funksiya emas — monetizatsiya mantig'i (freemium) va ketma-ketlik (qism/hamkorlik keyinroq), plyus kuratsiya chuqurligini qo'shish.
