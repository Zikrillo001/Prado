# Ochiq savollar — Dilshod bilan aniqlanadi

Loyihaning keyingi shakli aynan shu javoblarga bog'liq.

## Biznes / resurs
- [ ] Byudjet — loyihaga qancha mablag' ajrata oladi?
- [ ] Jamoa — hozir kim yordam beradi? Xodim yollay oladimi?
- [ ] Real raqamlar — oyiga nechta mashina, o'rtacha bitim qiymati, marja?
- [ ] Yuridik tuzilma — YTT / MChJ, STIR bormi?

## Mahsulot / qamrov
- [ ] Faqat premium yo'ltanmasmi, yoki keyin hamma mashinaga kengaymi?
- [ ] Dilshodning o'z mashinalari ham bo'ladimi (rasmiy do'kon), yoki faqat boshqalarniki?
- [ ] Club MVP'da bormi, yoki keyinroqmi?
- [ ] Ehtiyot qismlar qachon — boshidan yoki keyin?
- [ ] Geografiya — faqat Toshkentmi yoki butun O'zbekistonmi?

## Monetizatsiya
- [ ] Ko'tarish narxlari qanday (dinamik/sobit, qancha)?
- [ ] Diler paketlari bo'ladimi?

## Brend
- [ ] Nomi: Dilshod Prado / Prado Club / Dilshod_Prado?
- [ ] Til: o'zbek + rus (ikkalasi)mi?

## Sotuvchi
- [ ] Faqat jismoniy shaxslarmi, yoki dilerlar/avtosalonlar ham?
