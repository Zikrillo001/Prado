# Qarorlar jurnali

Qabul qilingan strategik qarorlar va sabablari.

| Qaror | Nima | Sabab |
|---|---|---|
| Model | Klassifayd vositachilik (mashina puli platformada emas) | Eskrou/qonun to'sig'idan qochish + OLX modeli; platforma faqat bog'laydi |
| Monetizatsiya | Freemium (bepul e'lon + pullik ko'tarish) | Raqib (Avtoelon/OLX) bepul; "har e'lon pullik" sotuvchini qochiradi |
| Pozitsiya | Tor premium nisha + ishonch + jamoa | Kaspi/Kolesa bilan generic jang yutqaziladi; farqlash shu yerda |
| Club | Strategiyaning markazi (yon funksiya emas) | Raqib nusxa ololmaydigan qal'a; regional pretsedent bor (Prado-Club 2008-) |
| Ishga tushirish | Dilshod auditoriyasini ko'chirish + o'z inventari bilan to'ldirish | Bring a Trailer modeli; sovuq startni ~63 ming auditoriya yechadi |
| Qism / hamkorlik | Keyinroq (P2) | Qamrov; Avtoelon'da allaqachon bor |
| MVP | Yadro + ishonch + Dilshod inventari + ko'tarish to'lovi | Modelni tez, arzon tekshirish |

## Muhim faktlar (research)
- Avtoelon.uz = Kolesa Group = Kaspi.kz (Markaziy Osiyo gigant).
- Kolesa: ~94% ulush KZ'da; Kolesa Zapchasti (qism) + Kolesa Finans (kredit) allaqachon bor.
- Bring a Trailer: blogdan boshlangan, jamoa -> marketplace, $1 mlrd+.
- O'zbekistonda premium yo'ltanmasga bag'ishlangan alohida platforma YO'Q — maydon bo'sh.
